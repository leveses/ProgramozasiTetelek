# ProgramozasiTetelek

## Elõkészítés
- 10 elemû egész számot tartalmazó tömb létrehozása
- A tömb elemei: 12, 7, 2, 4, 45, 99, 1, 8, 95, 50
- A tömb az összes metódusban látható legyen
- A TombKiiras() metódus létrehozása
  - Nem ad vissza értéket
  - Nincs paramétere 
  - A tömb elemeit vesszõvel elválasztva egy sorba a consolra írja
## Összegzés
- Osszegzes() metódus létrehozása
- Összegzés tétele a tömbön
- Eredmény kiírása a consolra
## Megszámlálás
- Megszamolas() metódus létrehozása
- Hány 3-al osztható elem van a tömb-ben
## Eldöntés
- Eldontes() metódus létrehozása
- Van-e a tömben 69?
## Kiválasztás
- Hányadik az 50-es?
## Keresés
- Van-e benne 50 és ha van hányadik?
## Maximum
- Kiírni a legnagyobb elemet
